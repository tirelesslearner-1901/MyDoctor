# MyDoctor
Patient Reviews and Feedback System implemented using MERN stack.

### Context
MyDoctor is a mini application built on MERN stack that collects patient reviews after a medical appointment. The program utilizes material UI CSS framework, mongoDB Atlas and Redux for state management.

A script contains the type of questions that the patient has to be asked. Sample data was also provided in JSON format with the patient, doctor, and appointment details, which were rendered to be utilized in the given script in order to create an user-engaging experience.

### Usage
Create the build of front-end folder by typing `npm run build` and once gets finished , come into the root folder and type `nodemon` in the terminal.The application starts running on `localhost:3001/`
